# Terceiro-Trab-AED

## Guide Lines
- Nomes de funções: lowerCamelCase;
- Nomes das estruturas: UpperCamelCase;
- Nomes de variáveis: lower_underline;
- Comentários no padrão doxygen contendo:
    - Pré-condições;
    - Pós-condições;
    - Brief, uma descrição resumida;
    - Parâmetros, se existirem;
    - Retorno, se existir.
